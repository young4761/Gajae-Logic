## 🔍 POA 작동 방식 파악

```
TradingView 차트
     │
     ↓ 전략 스크립트 실행 (dokang/POA/17 라이브러리)
     │
POA.bot.new("youngsoon", ...)
     │
     ↓ 조건 충족 시
     │
웹훅 메시지 생성 → Discord Webhook
     │
     ↓
POA 서버 (Vultr) POST /order
     │
     ↓ MarketOrder 파싱
     │
거래소 API 호출 (ccxt 또는 직접)
     │
     ↓
주문 실행 → Discord 알림
```
