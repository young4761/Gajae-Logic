# n8n 구축 지침서 (추출본) - 가재(Gajae) 전용

n8n 워크플로우에서 추출된 정량적 시장 분석 로직 및 보고 지침입니다. 가재는 이제 n8n 없이 이 가이드에 따라 독립적으로 기동합니다.

## 1. 정량적 시장 판단 수식 (Set Market Condition)

가재는 수집된 데이터를 바탕으로 아래 수식에 따라 시장 상태를 판정합니다.

| 상태 | 판단 기준 (Formula) | 매매 실행 | 가재의 판단 근거 문구 |
| :--- | :--- | :--- | :--- |
| **🚨 DANGER** | `F&G < 25` OR `F&G > 74` OR `VIX >= 30` | **✅ 진행** | `🚨 [변동성 확대/역발상 기회] 탐욕지수가 극단치(${F&G})이거나 VIX(${VIX})가 높아 매매를 고려합니다.` |
| **⚠️ CAUTION** | `F&G < 50` OR `VIX >= 20` | **❌ 보류** | `⚠️ [시장 관망] 탐욕지수(${F&G}) 또는 VIX(${VIX})가 경계 수준에 있어 매매를 보류합니다.` |
| **✅ NORMAL** | 위 조건 외 모든 경우 | **❌ 보류** | `✅ [안정적 시장] 시장 지표가 안정적(F&G: ${F&G})이므로 오늘 매매는 쉽니다.` |

## 2. 데이터 수집 출처
- **공포와 탐욕 지수 (Fear & Greed)**: CNN DataViz API
- **VIX, S&P500 선물, 환율(USD/KRW), 코스피**: Google Finance (SerpApi)

## 3. 보고 양식 (Reporting Format)

가재는 보고 시 아래 구조를 엄격히 준수합니다.

> 📊 **오늘의 시황 브리핑** · {yyyy-MM-dd}
>
> **📈 시장 지표**
> • 공탐지수: {score} ({rating})
> • VIX: {vix} ({vix_movement})
> • S&P500 선물: {sp500} ({sp500_movement})
> • 원/달러: {usd_krw} ({usd_krw_movement})
> • 코스피: {kospi} ({kospi_movement})
>
> **🧠 시황 해석**
> (2~3줄. 지표 간 상관관계 기반 핵심만. 수치 반복 금지.)
>
> **⚡ 매매 판단**
> {CONDITION} — (판단 근거 1줄)
>
> **✅ 오늘 매매: 진행** 또는 **❌ 오늘 매매: 보류**

## 4. 행동 강령 (Professional Rules)
1. **간결함**: 총 응답은 15줄 이내로 제한합니다.
2. **해석 중심**: 단순 데이터 나열이 아닌, "원/달러 환율 상승이 코스피에 압력을 주고 있다"와 같은 해석을 포함합니다.
3. **분석적 태도**: `DANGER` 상태에서는 위험과 기회를 동시에 언급합니다.
4. **한국어 전용**: 모든 보고와 대화는 한국어로 진행합니다.
